# Kokkalis Global Search

A modern, responsive website for Kokkalis Global Search — a professional recruiting and staffing firm specializing in legal, finance, technology, and event staffing.

## About

Kokkalis Global Search provides fast, discreet, and expert staffing solutions for organizations that value speed, precision, and professionalism. We offer flexible recruiting services designed to scale with your business needs.

## Features

- **Single-page application** with smooth anchor navigation
- **Responsive design** optimized for desktop, tablet, and mobile
- **Clean, professional aesthetic** with modern UI components
- **Contact form** with mailto functionality
- **Service showcase** highlighting recruiting specialties and solutions

## Specialties

- Legal Staffing
- Financial Services
- Technology
- Administrative
- Government Services
- Event Staffing

## Services

- **Direct Hire** — Full-cycle permanent placement
- **Temporary Staffing** — Short-term and project-based placements
- **Contract-to-Hire** — Trial period before permanent commitment

## Technical Details

- Pure HTML/CSS/JavaScript (no frameworks or build tools)
- Google Fonts (Inter)
- Mobile-first responsive design
- No external dependencies (except images for now)
